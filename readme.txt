Character Case Converter Program:
This C++ program converts a given character from uppercase to lowercase or vice versa.

Program Overview:

- Prompts the user to input a character
- Checks if the input character is an alphabet
- If the character is uppercase, converts it to lowercase
- If the character is lowercase, converts it to uppercase
- Displays the converted character

Features:

- User input for character
- Use of isalpha, isupper, tolower, and toupper functions for character checking and conversion
- Clear and informative output

Functions Used:

- isalpha(): checks if a character is an alphabet
- isupper(): checks if a character is uppercase
- tolower(): converts a character to lowercase
- toupper(): converts a character to uppercase

How to Run the Program:

1. Copy the provided C++ code into a .cpp file
2. Compile the program using a C++ compiler
3. Run the compiled program:
    - On Linux/macOS: ./character_case_converter
    - On Windows: character_case_converter.exe
4. Enter a character when prompted

Learning Objectives:

- Using character checking and conversion functions

- Basic program structure and syntax
